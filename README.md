# asd-egz

Podzial na kategorie. Kazde zadanie do odpowiedniej kategorii
Sortowania, struktury, grafy, prog. dynamiczne

Schemat nazywania:
- kolD_X_20YY_20YY_[Z]
- egz_X_20YY_20YY_[Z]

Gdzie
- D - numer kolokwium/termin egzaminu
- X - numer zadania
- YY - rok
- Z - tresc / czego dotyczy (skrotowo)
